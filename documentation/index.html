Installation

  1. Download the main package from ThemeForest.
  2. Extract the ZIP file.
  3. Open the html folder.
  4. Select your preferred demo.
  5. Upload files to your hosting server.
  6. Open index.html in browser.

How to Change Logo

Replace the logo image inside:

assets/images/logo.png

Or edit text logo in HTML:

<a class="logo">Kamakor</a>

How to Change Text Content

Open HTML file with code editor and edit text directly.

<h1>Build Your Digital Presence</h1>

Change to your own title.

How to Change Colors

Open CSS file:

assets/css/style.css

Find root variables:

:root{ --primary-color:#2563eb; --secondary-color:#111827; }

Color Presets

  • blue.css
  • purple.css
  • green.css
  • red.css
  • gold.css

Link preferred preset:

<link rel="stylesheet" href="assets/css/blue.css">

Dark Mode

Kamakor includes dark mode support. Add dark class:

<body class="dark-mode">
